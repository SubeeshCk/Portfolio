A high-performance, interactive personal portfolio website built to showcase my work as a Full Stack Developer.

The AI Experiment This project serves as a practical exploration of AI-Augmented Development. Built in collaboration with Antigravity, it demonstrates how developers can leverage advanced AI agents to accelerate production, refine code quality, and implement complex UI interactions with precision.

Built With:
Frontend: React, Vite
Styling: Tailwind CSS
Motion: Framer Motion, HTML5 Canvas
Collaboration: Antigravity (AI Agent)

Key Features:
Immersive Experience: Custom cursor and magnetic interactions.
Interactive UI: 3D tilt effects on project cards and a reactive starfield background.
Clean Code: Modular component structure optimized by AI-assisted reviews.
